# My dotfiles

## Setup
- Clone project
    ```bash
    git clone https://github.com/msal4/dotfiles.git $HOME/dotfiles
    ```
- Ensure you're in the dotfiles directory for the shell script to work properly
    ```bash
    cd dotfiles
    ```
- Link dotfiles
    ```bash
     ./setup # -f to overwrite any existing dot files if they exist
    ```
